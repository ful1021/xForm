# xForm
基于Vue2.x的自定义表单轮子。**当前处于早期开发阶段，所有`api`皆可能发生变动，请勿在生产环境使用。** 该项目灵感来源于以下两处：
- 钉钉审批管理中的表单设计器
- [formbuilder](https://github.com/dobtco/formbuilder)
